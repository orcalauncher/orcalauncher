Frequently Asked Questions

Is this launcher safe to download from?
Yes. All files are 100% checked before release. There are no hidden malware and each game is pre-installed for you. All you have to do is unzip and run.
403 forbidden error when trying to download a game?
If you get a 403 forbidden error that means you are trying to download more than 1 game at a time. UploadHaven currently only allows 1 active download per IP. So cancel any other downloads and only download one game at a time.
Can you play online?
Unless otherwise stated (usually in the game title or in the comments) all games are offline. It may be possible in some games, there are ways to do it, but you will have to research yourself since it differs per game, so we can't give you the specific answer here.
I'm having errors when trying to run the game?
Make sure your computer meets the minimum requirements. Crashes in-game, freezes and black screens are usually caused by low-end computers, outdated drivers, and missing files. If there’s a missing Dll or file, disable your antivirus/windows defender and check for a Redist or _CommonRedist folder, and be sure to install all the required software in those folders: DirectX, vcredist, DotNet, etc.
I'm having problems saving the game?
If you have issues saving a game, right click the exe and select “run as administrator”.
The .zip file is corrupt after download?
Winrar can have issues extracting some of the games, so switching to 7-zip should solve the problem.
I'm having problems with downloading the game?
If you get a 403 forbidden error that means you are trying to download more than 1 game at a time. UploadHaven currently only allows 1 active download per IP (Free UserS). So cancel any other downloads and only download one game at a time.
My download is very slow?
The Download Speed is Influenced By: Wether you are using the Fast Download option or not. Your internet plan (for example a user’s 30 MB/s download speed can manage to download from non-premium UploadHaven servers at around 5 MB/s). The number off people using your modem. The servers the file is hosted on. The number of people trying to download the file at the same time. If you can, use a lan cable for downloading (direct connection from the router to your device) especially for bigger games. Downloading files on wifi can lead to the corruption of the file itself, making it unusable. This means when your wifi drops from X amount of seconds/minutes and it’s unable to resume the download when the internet is back on.
My Anti-virus detects virusses?
Many anti-virus programs detect game files as a “virus” because the game files contained cracked files which make the game run. A lot of these anti-virus companies get paid to do this, that way you buy the game instead. To prevent this, disable your anti virus, then re-extract the .zip file and the game will run. Every game on this site was tested on our own computers, so you can be insured that it’s not a real virus. Anything your A.V may tell you, is a false positive.
